Recent Projects and Works

GABAY: Smart Walking Stick for Persons with Visual Impairment

The Gabay Smart Walking Stick was a thesis project prototype made and presented during the final year of college with the help of the Philippine Blind Union. The project was done with the aim of using contemporary technology that can help lead persons with visual impairment on independent journeys and analyze their surroundings for any hazards.

The project made use of an Arduino microcontroller, an ESP32-CAM, Ultrasonic Sensors, Gyroscope and Accelerometer Sensor, several motors, and a mobile application. With these, the device is made capable of detecting nearby obstructions for the user and also sending emergency messages automatically in times of accidents. With the use of Python and MySQL, the system can also recognize particular obstacles seen by the camera which then lets the user know about it. The object detection model was made by training a YoloV5 model in Google Colab and datasets are made in Roboflow

ARA: Augmented Reality Assistant

ARA is a project made that combines Augmented Reality, Internet of Things, and Data Science into one system. With the use of the mobile application, users can connect to the local home network which allows them to control devices connected to the local network via an Augmented Reality interface.

The ARA project uses an Arduino board to control small devices and also get data from connected sensors. This microcontroller then communicates with a C++ program that acts as a way to access the MSSQL Database. The AR app developed from the Unity Development Platform using C# can then access the database to communicate with the microcontroller remotely.

SmartLight: Home Lighting Monitoring Web Application

The SmartLight project is a collaborative effort which combines an Arduino microcontroller system with a web application. This microcontroller system is connected to the computer's MSSQL database and the web application displays the information stored in the database.

The lights are controlled by an Arduino using photoresistors designed to turn on when the area is dark and vice versa. The web application accessible is via desktop or mobile browsers can display an estimation on energy bill based on how long the lights are turned on.

Oil Price Prediction Data Science Project

The Price Prediction project was done as an academic project tasked to apply data science into a system. The project was developed using C++/CLI in Visual Studio 2022 as a C++ desktop application that processes CSV data regarding Petrol Prices to create a prediction of how much the price of petrol will be in the following years, months, or weeks using Average and Regression statistics.

Binance Market Position & Risk Calculator Java Application

With my interest in crypto trading, the position & risk calculator was done as a personal project designed to assist myself with low-timeframe and fast-paced cryptocurrency trading by instantly calculating risks and position sizes while taking into account fees in using the Binance Crypto Trading Platform. The application was developed with Java using the NetBeans IDE.

EZFX: MT4 Forex Lot Size & Risk Calculator

The EZFX is another project inspired by my personal interest in trading. For low-timeframe traders, every second counts. Calculating the right lot size takes a lot into consideration and will also take a lot of time which can cause missed trading opportunities for the trader.

The EZFX is an Expert Advisor program for MetaTrader 4, or MT4, made using MQL4. This program is designed to be used in lower timeframe trading in MT4 to instantly calculate the right lot size based on the set stop-loss levels, amount willing to be risked, profit ratio, and spreads. With this, traders only need to set a stop-loss level and the amount willing to be risked, then the order can be instantly placed.
